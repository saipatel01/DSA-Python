# DSA-Python

A collection of Data Structures and Algorithms (DSA) problems solved in Python.

## Arrays

This repository contains solutions to commonly asked array problems:

* Largest Element in an Array
* Second Largest Element in an Array
* Check if Array is Sorted
* Remove Duplicates from Sorted Array
* Reverse an Array
* Rotate Array Left
* Rotate Array Right
* Move All Zeros to End
* Maximum Consecutive Ones
* Missing Number in Array
* Single Number
* Monotonic Array
* Two Sum
* Buy and Sell Stock
* Contains Duplicate

## Goals

* Strengthen problem-solving skills
* Learn DSA patterns
* Prepare for coding interviews
* Practice Python programming
